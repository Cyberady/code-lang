# Code

> **Simple enough for beginners. Powerful enough for production.**

Code is a modern programming language focused on readability, consistency, and developer productivity.

It removes unnecessary syntax while keeping the power required for real-world software development.

---

# Philosophy

Code follows a few simple principles:

* Humans First
* One Obvious Way
* Less Syntax
* Readability Over Cleverness
* Batteries Included
* Production Ready

---

# Features

* Simple variable declarations
* Immutable constants
* One numeric type
* Unicode strings
* String interpolation
* Automatic function returns
* Arrays
* Objects
* Readable conditionals
* Clean loop syntax

---

# Example

```code
const PI = 3.14159

func area(radius) {

    PI * radius * radius

}

radius = 10

result = area(radius)

print("Area = {result}")
```

---

# Conditionals

```code
if age >= 18 {

    print("Adult")

} else {

    print("Minor")

}
```

---

# Loops

```code
for user in users {

    print(user)

}
```

```code
for i in range(10) {

    print(i)

}
```
